## 🌷 About

Some thoughts stay hidden forever — not because they don't matter, but because there was never a safe place to say them.

**Things I Never Said** is a soft, anonymous space for honest expression. No sign-ups. No usernames. No judgment. Just words that needed somewhere to go.

---

## ✨ Features

- 📝 Post anonymous confessions instantly — no account needed
- 🌍 All confessions publicly visible to the world
- 🔒 Fully anonymous — zero identity required
- 🌸 Soft pastel minimalist UI designed for emotional comfort
- 📱 Fully responsive across all devices
- ⚡ Fast, modern full-stack architecture

---

## 🎨 Design Philosophy

The interface is intentionally quiet, gentle, and distraction-free. Every design decision prioritizes emotional comfort over engagement mechanics.

- **Minimalism** — nothing unnecessary, nothing loud
- **Soft pastel aesthetics** — calm colors that feel safe
- **Clean typography** — easy to read, easy to feel
- **No distractions** — no likes, no followers, no algorithms

## 🚀 Getting Started

### Prerequisites

- Node.js `18+`
- npm or yarn

### Setup

```bash
# 1. Clone the repository
git clone https://github.com/your-username/things-i-never-said.git
cd things-i-never-said

# 2. Install frontend dependencies
cd frontend
npm install
npm run dev

# 3. Install backend dependencies (new terminal)
cd backend
npm install
npm run start:dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
