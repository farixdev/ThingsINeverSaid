# 🌸 Things I Never Said

> A place for the words people never had the courage to say.

**Things I Never Said** is an anonymous confession platform built with **Next.js** and **NestJS** where users can freely share thoughts, regrets, feelings, and emotions they never said out loud. Designed with a soft minimalist aesthetic, the platform creates a calm and safe space for honest expression without identity or judgment.

---

## ✨ Features

- 📝 Post anonymous confessions
- 🌍 Publicly visible confessions
- 🎨 Soft minimalist pastel UI
- ⚡ Modern full-stack architecture
- 🔒 No accounts or identity required
- 💬 Real thoughts from real people
- 📱 Responsive design
- 🌸 Clean and calming user experience

---

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/things-i-never-said.git
```

### Navigate into the project

```bash
cd things-i-never-said
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```
